# CAD Design

This folder contains the SolidWorks design files for the hexapod robot.

## 🔧 Overview

The hexapod robot was designed as a 6-legged mechanical system with joint-based leg mechanisms to enable locomotion.

The design focuses on functionality, joint movement, and compatibility with simulation environments.

---

## ⚙️ Features

- 6-legged robot structure  
- Joint-based leg mechanism  
- Modular design for easy modification  
- Designed for simulation in NVIDIA Isaac Sim  

---

## 🧩 Design Approach

The CAD model was simplified by excluding small components such as screws and bearings.

This was done to:
- Reduce computational complexity  
- Improve simulation performance  
- Maintain focus on functional kinematics  

---

## 📌 Notes

- The design prioritizes movement and testing over manufacturing-level detailing  
- Components are aligned for proper joint motion and simulation accuracy
