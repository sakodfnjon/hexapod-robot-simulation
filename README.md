# Hexapod Robot Simulation

A hexapod robot designed and simulated using SolidWorks and NVIDIA Isaac Sim, with manual control and reinforcement learning experiments for locomotion.

---

## 🔧 Overview

This project focuses on the design, simulation, and control of a 6-legged (hexapod) robot.  
The robot was modeled in SolidWorks and tested in NVIDIA Isaac Sim to study movement, stability, and control strategies.

## ⚙️ Features

- 6-legged robot designed in SolidWorks  
- Joint-based leg mechanism for locomotion  
- Simulation in NVIDIA Isaac Sim  
- Manual control system for testing movement  
- Reinforcement Learning experiments (~4000 iterations)  

---

## 🎮 Control System

Implemented a manual control interface to dynamically adjust joint angles and test movement behavior in simulation.  
This allowed testing of different movement patterns and understanding of joint coordination before automation.

---

## 🧠 Reinforcement Learning

Initial experiments were conducted using reinforcement learning to enable the robot to learn locomotion through trial and error.

- ~4000 training iterations completed  
- Partial movement achieved  
- Further tuning required for stability  

---

## 🛠️ Tech Stack

- SolidWorks (CAD Design)  
- Python (Control & Logic)  
- NVIDIA Isaac Sim (Simulation)  

---

## 🧩 Design Approach

The CAD model was simplified by excluding small components such as screws and bearings to optimize simulation performance while maintaining functional kinematics.

---

## 🚀 Future Improvements

- Implement stable walking gait (tripod gait)  
- Apply inverse kinematics for better control  
- Improve reinforcement learning performance  
- Deploy on real hardware (ESP32 / servo-based system)  

---

## 📌 Notes

This project is focused on learning robotics concepts by integrating mechanical design, simulation, and control systems.
