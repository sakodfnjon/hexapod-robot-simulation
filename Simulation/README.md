# Simulation

This folder contains the simulation setup for the hexapod robot using NVIDIA Isaac Sim.

---

## 🔧 Overview

The hexapod robot is imported into a physics-based simulation environment to test movement, joint behavior, and control strategies before real-world implementation.

---

## ⚙️ Features

* Physics-based simulation in NVIDIA Isaac Sim
* Real-time joint control and visualization
* Support for both manual control and reinforcement learning workflows

---

## 🧩 Purpose

The simulation environment is used to:

* Validate robot movement and kinematics
* Analyze joint coordination and stability
* Test control strategies safely before hardware deployment
* Serve as a platform for reinforcement learning experiments
